## Android-compose-mvvm-movie-app



A Kotlin-based Android app built with MVVM, Jetpack Compose, Coroutines, and Flow. 
Fetches movie data from a remote API and shows state handling in Compose.

## Features
- Movie list from API
- Favorites / Watchlist
- Search movies
- Loading & Error UI
- Offline caching

## What I Learned
- MVVM with Repository & ViewModel
- Async API calls with Coroutines & Flow
- Compose UI + state management
- Dependency Injection (Hilt)
